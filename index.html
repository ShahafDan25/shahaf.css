Welcome !

Welcome to shahaf.css !

shahaf.css is a small, simple, compact, and convenient-for-use CSS library for the intermediate developer!

If you're a new developer, this library is exactly for you, providing quick CSS styling short cuts by the assignment of hirarchy class names to HTML elements.

Our objective is to make CSS styling and programming much more conveniet and easy for all programmers and developers, by saving a great deal of time and assisting you, the developer, to avoid repeatitive CSS code.

To start, install this file and place it within your project's folder.

-- IZES

.borderize

Surround your shahaf.css objects and elements with a simple border

.borderize element

.rightize

Push all of your elements in a container to the right (horizontally)

.centrize

Push all of your elements in a container to the center (horizontally)

.leftize

Push all of your elements in a container to the left (horizontally)

.blockize

Display a conatiner as a centered block, in full width in its parents container

.blockize element

.inlinize

Display ell elements in the same parents container with the .inlinize class, in line, overriding pervious CSS definitions. The inlinized items will only fit inline however many elements it can.

.shadowize

Assign a shadow for you object to create an illusion of a floating element.

.roundize & .circlize

Round the corners of you containers and elements by using the the .roundize and .circlize shahaf.css classes and will override any previous assignments!

.roundize will give your object the average corner-curvature.
.roundize-none will remove any curvature from your object's / container's corners
.roundize-low will give your object's / container's corners a rather low curvature rate.
.roundize-medium will give your object's / container's corners a rather medium curvature rate.
.roundize-high will give your object's / container's corners a rather high curvature rate.
.circlize will completely curve the corners of your element / container.
